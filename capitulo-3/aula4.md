# Aula4

* Exibir a descrição de uma tabela:
  * `DESCRIBE pet;`

```
SELECT * FROM pet;
```

* Atualizando dados de uma tabela:

```
UPDATE pet SET breed = 'PUG' WHERE ID = 2;
```

* Deletando dados de uma tabela:

```
DELETE FROM pet WHERE name = "Pico de Gato";
```
